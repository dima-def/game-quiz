Такие языки как: Python, Java, LISP используются для машинного обучения! Очень хороший выбор для прогроммистов которые хотят работать и развивать ИИ

Также есть высокоуровневые языки которые будут актуальные еще очень долго - например C, C++, JavaScript и т.д

Еще есть браузерные языки: JavaScript, HTML и CSS.

После выбранного языка программирования у большинства новичков возникает вопрос: "А где же мне писать код?"

Что ж есть несколько вариантов:

1) Visual Studio Code

2) Visual Studio

3) PyCharm

4) А также есть онлайн компиляторы(для всех языков)

Краткая характеристика:

Visual Studio Code - один из лучших приложения для написания программ. Позволяет легко писать, форматировать и редактировать код на разных языках. С его помощью можно быстро создать проект и структуру файлов в нем, он подсвечивает синтаксис кода и помогает автоматически править ошибки. В нем есть возможности для отладки и запуска кода на некоторых языках.

P.S А также в VS Code можно скачивать разные дополнения сразу из приложения, а не через браузер!

Visual Studio - линейка систем разработки программного обеспечения от компании Microsoft. В своем составе имеют интегрированную среду разработки (IDE) и ряд других инструментов.

PyCharm - платное приложение для кодинга. Интегрированная среда разработки на языке Python, созданная компанией JetBrains. Среди питонистов эта IDE вторая по популярности после редактора кода Visual Studio Code: как основную её используют 31% разработчиков. В PyCharm есть все инструменты, чтобы писать, отлаживать и тестировать код.

Нужные ссылки:

Python - высокоуровневый язык программирования, отличающийся эффективностью, простотой и универсальностью использования. Он широко применяется в разработке веб-приложений и прикладного программного обеспечения, а также в машинном обучении и обработке больших данных.

С++ - компилируемый, статически типизированный язык программирования общего назначения. isocpp.org (англ.) Поддерживает такие парадигмы программирования, как процедурное программирование, объектно-ориентированное программирование, обобщённое программирование.

Java - многоплатформенный, объектно-ориентированный и сетецентрический язык, который сам по себе может использоваться как платформа. Это быстрый, безопасный и надежный язык программирования для всего: от мобильных приложений и корпоративного ПО до приложений для работы с большими данными и серверных технологий.

JavaScript - интерпретируемый и мультипарадигменный язык программирования. Инструмент разработчика, который обычно используется для написания веб-сайтов (frontend- и backend-частей). Может быть задействован при мобильной разработке. JavaScript является языком программирования высокого уровня.
